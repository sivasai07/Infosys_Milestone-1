# E-Commerce Recommendation Engine (Hybrid Approach)

## Milestone 1: Data Collection & Preprocessing

This project implements the first milestone of a hybrid recommendation engine system.

### Completed Tasks:
- Dataset selection from Kaggle
- Data cleaning and preprocessing
- Handling missing values
- Removing invalid IDs
- Cleaning product image links
- Creating user-item interaction matrix
- Initial EDA analysis

### Tech Stack:
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

### Project Structure:
- preprocess_data.py → Data cleaning logic
- main.py → EDA and validation
- data/ → Dataset
- requirements.txt → Dependencies
